# 🏭 Manufacturing Data Analysis using Apache Spark

## 📌 Project Overview

This project demonstrates large-scale manufacturing data analysis using **Apache Spark (PySpark)**. It covers the complete Spark workflow starting from Spark initialization to advanced DataFrame operations, Spark SQL, RDD transformations, caching, persistence, and analytical queries on manufacturing production data.

The project is implemented as part of an academic case study to understand distributed data processing and big data analytics using Apache Spark.

---

## 🚀 Objectives

- Initialize and configure Apache Spark
- Load manufacturing datasets into Spark
- Perform RDD transformations and actions
- Implement Key-Value RDD operations
- Apply persistence and caching techniques
- Perform DataFrame transformations
- Execute Spark SQL queries
- Analyze production efficiency
- Evaluate machine utilization
- Generate manufacturing insights
